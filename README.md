# Personal Portfolio (Ongoing Project)

[![React](https://img.shields.io/badge/React-17.0-blue?logo=react&logoColor=white)](https://reactjs.org/) [![Vercel](https://img.shields.io/badge/Deployed%20on-Vercel-000000?logo=vercel&logoColor=white)](https://vercel.com/)

## Overview
This is my **personal portfolio website** built using **React.js**.  
It showcases my skills, projects, and experience in software development and IT.  
The portfolio is fully **responsive**, **interactive**, and includes smooth **animations** for a modern user experience.  

This project is **ongoing**, and I continuously update it with new projects, features, and improvements.

**Live Demo:** [https://sakulajayarathne.vercel.app/](https://sakulajayarathne.vercel.app/)

---

## Features
- **Home, About, Projects, and Contact sections**  
- **Smooth scrolling** navigation with active link highlighting  
- **Responsive design** for mobile, tablet, and desktop  
- **Interactive animations** using custom React hooks  
- **Contact form** for users to reach out  
- Showcase of **projects with descriptions, images, and links**

---

## Tech Stack
- **Frontend:** React.js, HTML5, CSS3  
- **Styling:** CSS  
- **Animations:** React Hooks / CSS transitions
